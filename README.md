## 🚀 Install

Install it locally in your project

```bash
# npm
npm install next-define

# yarn
yarn add next-define

# pnpm
pnpm install next-define
```

## 🦄 Usage

```typescript
// ESM
import {} from "next-define";

// CommonJS
const {} = require("next-define");
```
